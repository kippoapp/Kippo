# Kippo — Sistema de Kiosko para Campamentos

> Gestión de ventas, saldos y stock para campamentos y actividades de tiempo libre. Rápido, intuitivo y pensado para monitores.

---

## ¿Qué es Kippo?

Kippo es una aplicación web diseñada para gestionar el kiosko de un campamento de forma sencilla y sin complicaciones. Permite controlar el saldo de cada acampado, registrar ventas en segundos, gestionar el stock y obtener estadísticas en tiempo real — todo desde cualquier dispositivo, sin instalar nada.

Cada cliente recibe su propio entorno privado, con su base de datos independiente y su link exclusivo. Sin interferencias entre campamentos.

---

## ¿Para quién es?

- Campamentos de verano y tiempo libre
- Colonias y escuelas de naturaleza
- Cualquier actividad con kiosko y sistema de saldo prepago

---

## Funcionalidades principales

**Para el monitor**
- Selección rápida de acampado por grupo
- Registro de ventas en pocos toques
- Visualización del saldo disponible en tiempo real
- Soporte para packs y ofertas combinadas

**Para el encargado / director**
- Panel completo de gestión con 8 secciones
- Control de saldos, grupos, monitores y acampados
- Gestión de productos, categorías y stock (kiosko y almacén)
- Historial completo de ventas y anulaciones
- Estadísticas: recaudación por día, ranking de ventas, productos más vendidos
- Exportación de datos a Excel
- Importación masiva de acampados, grupos y monitores desde plantilla Excel
- Tema oscuro y claro

**Sistema**
- Sincronización en tiempo real entre dispositivos vía Firebase
- Funciona como app instalable en móvil (PWA)
- Diseño adaptado a móvil y escritorio
- Sin necesidad de instalación — funciona desde el navegador

---

## Cómo funciona

1. El cliente recibe su link privado personalizado
2. Los monitores acceden desde su móvil y registran ventas al instante
3. El encargado supervisa todo desde el panel con su PIN de acceso
4. Los datos se sincronizan automáticamente entre todos los dispositivos

---

## Modelo de servicio

- **Pago único** por configuración e implementación
- **Cuota mensual** que incluye actualizaciones y soporte técnico
- Cada campamento tiene su entorno completamente independiente
- Las actualizaciones se despliegan automáticamente sin intervención del cliente
